User opens the web app and uses search feature to search for a restaurant (or another type of location/business). App returns a list of restaurants that best match the search. There will also be an option to have search results displayed on a heatmap (see heatmap story). User selects the restaurant from list or heatmap. App will display real time estimated COVID risk at different times of day at this restaurant and relative COVID risk with restaurants nearby. User will also have the option to show the restaurant on a heatmap (see heatmap story).
